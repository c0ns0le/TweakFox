Customise the quickdrag extension
---------------------------------

Objective:  Customise the quickdrag extension according to my preferences.

Important: The latest version at time of writting is 2.1.3.23.  This version contains adware.  The addons provides the user with the option to disable the adware and marketing.  However, the addon secretly re-enables the addon without informing the user.
Consequently, I recommend downloading version 2.1.3.21 (which does not contain any adware) and updating it to work with any version of the Firefox browser.  Version 2.1.3.21 is available at:
https://addons.mozilla.org/en-US/firefox/addon/quickdrag/versions/

The steps to enable this addon to work with any current or future version of the Firefox browser are detailed below.

1) Unzip the extension.

2) Edit the file /defaults/preferences/quickdrag.js

3) Set the following preferences:

* pref("extensions.quickdrag.loadLinkInFG", false);
* pref("extensions.quickdrag.loadSearchInFG", false);
* pref("extensions.quickdrag.downloadImages", true);
* pref("extensions.quickdrag.linkOpenOverride", true);
* pref("extensions.quickdrag.handleImageDrags", true);
* pref("extensions.quickdrag.handleNonImageDrags", true);
* pref("extensions.quickdrag.marketing", false);
	
4) Save the file and repackage the xpi extension.
